## Release Note

### 1.0.0（20220601）

1. 聚合 [国家法律法规数据库](https://flk.npc.gov.cn)，[最高人民法院](https://www.court.gov.cn) 网站数据
2. 离线使用
3. 全文，片段法条收藏
4. 文本分享
5. 层级目录
6. 高亮检索

### 1.1.0

* [ ] 项目数据以 [git subtree](https://lwn.net/Articles/235109/) 方式依赖
* [ ] 应用手动更新数据源选项
* [ ] 数据按照法条解析
* [ ] 目录可定位，以及流畅的滚动效果
* [ ] 分享内容生成图片
* [ ] 应用上架 F-Droid

### 2.0.0

* [ ] 黑夜模式
* [ ] 文字大小调整
* [ ] 行间距调整
* [ ] 法条间距调整
* [ ] 动态适配不同长度文章标题
